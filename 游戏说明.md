# 飞机大战游戏说明

## 游戏简介
这是一个使用Python和Pygame开发的经典飞机大战游戏。玩家控制一架飞机，躲避敌机并射击消灭它们。

## 游戏功能
- 玩家控制飞机移动和自动射击
- 敌机随机生成并移动
- 碰撞检测系统
- 分数和生命值显示
- 爆炸特效
- 游戏结束和重新开始功能

## 操作方法
- **方向键**：控制飞机移动（上、下、左、右）
- **自动射击**：玩家飞机会自动发射子弹
- **ESC键**：退出游戏
- **R键**：游戏结束后重新开始

## 游戏规则
- 击中敌机获得10分
- 被敌机撞击减少20点生命值
- 生命值降至0时游戏结束

## 运行要求
- Python 3.6+
- Pygame 2.0+
- 游戏包含基础音效系统

## 运行方法
```bash
python plane_game.py
```

## 游戏音效
- 射击音效：发射子弹时播放
- 爆炸音效：击中敌机时播放
- 被击中音效：玩家被敌机撞击时播放
- 游戏结束音效：生命值归零时播放
