# 唐诗网站数据结构规划

## 网站功能需求

### 1. 分类浏览
- 按作者分类
- 按卷数分类  
- 按主题分类（关键词）
- 按朝代分类

### 2. 搜索功能
- 全文搜索（标题、作者、内容）
- 高级搜索（多条件组合）
- 关键词联想

### 3. 数据展示
- 诗歌详情页面
- 作者信息页面
- 相关诗歌推荐

## 数据结构优化

### 1. 主数据文件
```json
{
  "poems": [
    {
      "id": "001-001",
      "title": "诗歌标题",
      "author": "作者",
      "volume": "卷数",
      "number": "编号",
      "paragraphs": ["诗句1", "诗句2"],
      "biography": "作者简介",
      "keywords": ["关键词1", "关键词2"],
      "dynasty": "朝代"
    }
  ]
}
```

### 2. 索引文件
- 作者索引
- 卷数索引  
- 关键词索引
- 朝代索引

## 前端技术栈建议
- HTML5 + CSS3 + JavaScript
- 可选框架：Vue.js / React
- 搜索库：Fuse.js / Lunr.js
- UI框架：Bootstrap / Tailwind CSS
